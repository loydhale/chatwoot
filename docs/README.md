# DeskFlow Documentation

This directory contains documentation for DeskFlow.

## Contents

- `/screenshots/` - Product screenshots for README and marketing
- `/deployment/` - Deployment guides (coming soon)
- `/api/` - API documentation (coming soon)

## Adding Screenshots

Place screenshots in the `/screenshots/` directory with descriptive names:
- `dashboard-placeholder.png` - Main agent dashboard
- `widget-placeholder.png` - Chat widget on a website
- `atlas-placeholder.png` - Atlas AI assistant in action

Recommended dimensions:
- Dashboard: 1280x720 or 1920x1080
- Widget: 400x600
- Marketing: 1200x630 (social sharing)
